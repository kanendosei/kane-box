kane-box is an Open Source Project, created to provide configuration tools, images and software which would run as a router or appliance in home or corporate networks.

The project, started in 2010, is focused on delivering C and Lisp-based source code to improve the state of Network Security.

In addition, there are a number of Lisp and newLisp snippets (and gists, if that is a word) in the github repository for doing all sorts of networking, statistical and other functions on a high-speed, high-traffic environment.
